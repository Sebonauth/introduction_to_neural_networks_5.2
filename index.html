Understanding Each Layer

Let's start with the Input Layer - the network's gateway for data.

The input layer is special because it's the only layer that doesn't perform any computations. It simply receives your data and passes it forward.

Input Layer

The first layer of a neural network that receives raw data. Each node represents one feature of your input data.

For example, in our previous marble classifier, the input layer had two nodes:

  • One for mass
  • One for diameter

Moving on to Hidden Layers - this is where the magic happens!

Hidden layers are what give neural networks their power to learn complex patterns. Each node in a hidden layer represents a learned feature that the network has discovered.

Why It Matters

The more hidden layers you have, the more complex patterns your network can learn. This is why we call networks with many hidden layers "deep" neural networks.

Finally, we reach the Output Layer - where your network makes its predictions.

The output layer's structure depends on your task:

  • For binary classification: One output node
  • For multiclass classification: Multiple nodes (one per class)
  • For regression: One node for each predicted value

Stop & Think

In a network designed to recognize handwritten digits (0-9), how many nodes would you expect in the output layer?

You'd need 10 output nodes - one for each possible digit!
